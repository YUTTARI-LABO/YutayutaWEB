― 主なコマンドの紹介 ―

― Introduction of main commands ―

デフォルトのプレフィックス(接頭辞)は y! です。
⧫ の記号が記載コマンドまたは記載されている箇所を実行するには権限が必要です。
<>で囲まれている所は必須項目。　[]で囲まれている所は必須ではない項目。

y!help...ゆたゆたのヘルプを表示します。
y!lang...現在のサーバー内でのゆたゆたの言語を表示します。
⧫ y!setlang <言語>...サーバー内でのゆたゆたの言語を設定します。
y!prefix [新しいプレフィックス(接頭辞)]...現在のサーバー内の接頭辞を表示または ⧫サーバー内の接頭辞を設定⧫ します。
⧫ y!command <enable または disable> <コマンドの名前>...そのコマンドをサーバー内で有効化(enable)または無効化(disable)します。
⧫ y!requiredrole <コマンドの名前> <none または ロール>...サーバー内でそのコマンドを実行するために必要なロールを設定します。
⧫ y!news...コマンドを入力したチャンネルでゆたゆたのお知らせを受け取ります。

The default prefix is y!
Permission is required to execute commands marked with the ⧫ symbol.
<> are required fields. [] are not required fields.

y!help...View Yutayuta's help.
y!lang...Displays the language of the current Yutayuta in the server.
y!setlang <Language>...Sets the language of Yutayuta in the server.
y!prefix [New Prefix]...Displays the prefix in the current server or ⧫ Sets the prefix in the server ⧫.
⧫ y!command <enable or disable> <Command Name>...Enable or disable that command in the server.
⧫ y!requiredrole <Command Name> <none or Role>...Set the role required to run that command in the server.
⧫ y!news...Receive notifications from YutaYuta on the channel where you entered the command.
